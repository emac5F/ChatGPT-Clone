# ChatGPT Clone - Vanilla JS Edition

Este es un proyecto de práctica diseñado para demostrar habilidades en el desarrollo de interfaces modernas utilizando **HTML5**, **CSS3 (Custom Properties)** y **JavaScript (Vanilla)**. El proyecto se centra en la creación de una experiencia de usuario fluida con soporte nativo para temas claro y oscuro.

## 🚀 Características

- **Diseño Responsivo:** Interfaz adaptada para dispositivos móviles y de escritorio.
- **Dark Mode Persistente:** Implementación de cambio de tema utilizando `data-attributes` y almacenamiento en `localStorage`.
- **Arquitectura de Código:** Lógica organizada mediante el patrón de diseño de objetos para una mejor mantenibilidad.
- **UI Moderna:** Burbujas de chat dinámicas y scroll automático.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica y uso de `<template>` para renderizado dinámico.
* **CSS3:** Layouts con Flexbox, Grid y variables CSS para el manejo de temas.
* **JavaScript:** Manipulación del DOM, manejo de eventos y almacenamiento local.

## 📦 Instalación y Uso

1.  Clona este repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/chatgpt-clone.git](https://github.com/tu-usuario/chatgpt-clone.git)
    ```
2.  Abre el archivo `index.html` en tu navegador preferido.
